# removeOverlay(filter?)
`removeOverlay` remove overlay.

## Reference {#reference}
<!-- @include: @/@views/api/references/instance/removeOverlay.md -->

### Parameters {#parameters}
- `filter` Filter conditions.
  - `id` Overlay id.
  - `name` Overlay name.
  - `groupId` Group id.
  - `paneId` Pane id.

### Returns {#returns}
`removeOverlay` returns `boolean` .
