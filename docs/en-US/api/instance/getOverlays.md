# getOverlays(filter?)
`getOverlays` get overlay information.

## Reference {#reference}
<!-- @include: @/@views/api/references/instance/getOverlays.md -->

### Parameters {#parameters}
- `filter` Filter conditions.
  - `id` Overlay id.
  - `name` Overlay name.
  - `groupId` Group id.
  - `paneId` Pane id.


### Returns {#returns}
`getOverlays` returns `Overlay[]` .
