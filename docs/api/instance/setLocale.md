# setLocale(locale)
`setLocale` 设置图表的语言。

## 参考 {#reference}
<!--@include: @/@views/api/references/instance/setLocale.md-->

### 参数 {#parameters}
- `locale` 语言名称，已内置 `zh-CN` 和 `en-US` 。

### 返回值 {#returns}
`setLocale` 返回 `undefined` 。
