# getOverlays(filter?)
`getOverlays` 获取覆盖物信息。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/getOverlays.md -->

### 参数 {#parameters}
- `filter` 过滤参数。
  - `id` 覆盖物id。
  - `name` 覆盖物名称。
  - `groupId` 编组id
  - `paneId` 窗口id。


### 返回值 {#returns}
`getOverlays` 返回 `Overlay[]` 。
