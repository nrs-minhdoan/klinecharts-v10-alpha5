# setThousandsSeparator(thousandsSeparator)
`setThousandsSeparator` 设置图表千分符配置。

## 参考 {#reference}
<!--@include: @/@views/api/references/instance/setThousandsSeparator.md-->

### 参数 {#parameters}
- `thousandsSeparator` 千分符配置。
  - `sign` 千分符符号。
  - `format` 自定义千分符格式化。

### 返回值 {#returns}
`setThousandsSeparator` 返回 `undefined` 。
