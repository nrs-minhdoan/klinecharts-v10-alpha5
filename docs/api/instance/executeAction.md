# executeAction(type, data)
`executeAction` 执行图表动作。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/executeAction.md -->

### 参数 {#parameters}
- `type` 类型，仅支持 `onCrosshairChange` 。
- `data` 数据。

### 返回值 {#returns}
`executeAction` 返回 `undefined` 。
