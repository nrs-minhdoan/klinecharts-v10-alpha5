# setDecimalFold(decimalFold)
`setDecimalFold` 设置图表小数折叠配置。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/setDecimalFold.md -->

### 参数 {#parameters}
- `decimalFold` 折叠配置。
  - `threshold` 折叠阈值。
  - `format` 自定义折叠格式化。

### 返回值 {#returns}
`setDecimalFold` 返回 `undefined` 。
