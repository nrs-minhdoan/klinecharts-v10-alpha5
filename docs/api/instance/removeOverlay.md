# removeOverlay(filter?)
`removeOverlay` 移除覆盖物。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/removeOverlay.md -->

### 参数 {#parameters}
- `filter` 过滤参数。
  - `id` 覆盖物id。
  - `name` 覆盖物名称。
  - `groupId` 编组id
  - `paneId` 窗口id。


### 返回值 {#returns}
`removeOverlay` 返回 `boolean` 。
