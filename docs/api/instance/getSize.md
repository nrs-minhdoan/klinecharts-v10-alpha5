# getSize(paneId?, position?)
`getSize` 获取图表的尺寸。

## 参考 {#reference}
<!--@include: @/@views/api/references/instance/getSize.md-->

### 参数 {#parameters}
- `paneId` 窗口 id 。
- `position` 位置，支持 `root` ， `main` 和 `yAxis` 。

### 返回值 {#returns}
`getDom` 返回一个包含尺寸信息的 `Bounding` 对象或者 `null` 。
